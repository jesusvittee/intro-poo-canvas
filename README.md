# 🎨 Generación de Círculos con Canvas

## 📌 Descripción

Este proyecto utiliza **HTML5 Canvas y JavaScript** para generar y dibujar círculos dentro de diferentes áreas de dibujo.  
Se implementa **Programación Orientada a Objetos (POO)** mediante una clase llamada `Circle`, la cual permite crear círculos con diferentes propiedades como posición, tamaño, color y texto.

El programa muestra tres ejemplos:

- Un círculo centrado
- Un círculo generado en posición aleatoria
- Múltiples círculos con tamaños y colores diferentes

Además, se controla que los círculos **no se salgan del área del canvas**.

---

## ⚙️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Canvas API

---

## 🧠 Conceptos aplicados

- Programación Orientada a Objetos
- Clases en JavaScript
- Uso de `Math.random()` para generar valores aleatorios
- Uso de arreglos para almacenar objetos
- Manipulación del contexto gráfico de Canvas

---

## 🚀 Funcionamiento

1. Se obtienen los elementos `canvas` del HTML.
2. Se define su tamaño y color de fondo.
3. Se crea la clase `Circle`.
4. Se generan círculos con posiciones y tamaños aleatorios.
5. Los círculos se dibujan usando el método `draw()`.
